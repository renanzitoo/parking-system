# Parking Project

A simple parking management system developed in **C# with .NET**,
focused on practicing **clean code**, **separation of
responsibilities**, and **unit testing**.\
The project allows adding, removing, and listing vehicles while
calculating parking fees.

## ğŸš€ Installing app code

If you have **SSH configured**, run:

``` bash
git clone git@github.com:renanzitoo/parking-project.git
```

If you **do not have SSH configured**, run:

``` bash
git clone https://github.com/renanzitoo/parking-project.git
```

After cloning the repository, access the project folder:

``` bash
cd parking-project
```

Restore dependencies:

``` bash
dotnet restore
```

Run the application:

``` bash
dotnet run --project ParkingProject
```
